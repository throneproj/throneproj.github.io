Throne поддерживает два режима прокси:

## Системный прокси (System Proxy)

Направляет трафик всей системы через прокси путем изменения системных настроек прокси-сервера. Подходит для таких приложений, как веб-браузеры.

## Режим TUN

Создает виртуальный сетевой интерфейс для обеспечения глобального проксирования. Весь сетевой трафик приложений будет проходить через прокси, включая программное обеспечение, которое не поддерживает настройки прокси.

> Рекомендуется использовать режим TUN для более полного проксирования.

> Примечание: При включении режима TUN отключите системный прокси; они не могут быть активированы одновременно.

## Импорт подписки

### Способ 1: Прямая вставка

Нажмите `Ctrl+V` прямо в главном окне, чтобы вставить ссылку на подписку.

> После импорта группе по умолчанию присваивается имя `Default`. Щелкните правой кнопкой мыши по группе и выберите `Обновить подписку` (Update Subscription) (горячая клавиша: `Ctrl+U`).

### Способ 2: Импорт через настройки

1. Перейдите в `Настройки` (Settings) → `Группы` (Groups)
2. Создайте новую группу и введите имя
3. Измените тип на `Подписка` (Subscription)
4. Вставьте ссылку на подписку в поле URL
5. Нажмите ОК для сохранения
6. Щелкните правой кнопкой мыши по этой группе и выберите `Обновить список` (Refresh List)

Если ссылка верна, список прокси будет загружен и отображен.

## Проверка пинга (Latency)

1. Выделите весь список, затем щелкните правой кнопкой мыши
2. Выберите `URL-тест выбранных элементов` (URL Test Selected Items) (горячая клавиша: `Ctrl+Shift+S`)

![URL-тест](/images/configuration/4.png)

3. Выберите узел с наименьшей задержкой, щелкните правой кнопкой мыши и выберите `Включить` (Enable) или просто нажмите `Enter`

## Включение прокси
### Настройка правил
Независимо от того, используете ли вы режим TUN или системный прокси, необходимо настроить правила маршрутизации (на примере материкового Китая):

1. Нажмите `Маршруты` (Routes) → `Скачать профиль настроек` (Download Config Profile)
2. Выберите `Bypass China` + `Bypass China Blocked`
3. Нажмите `Маршруты` (Routes) → `Настройки маршрутизации` (Routing Settings) → `Маршруты` (Routes)
4. Удалите правила маршрутизации по умолчанию, оставив только два только что скачанных правила

![Настройки маршрутизации](/images/configuration/5.png)

### Запуск прокси

> Рекомендуется режим TUN.

1. Сначала запустите узел
2. Затем выберите режим прокси (TUN или системный прокси)

> Примечание: Если вы используете режим TUN, пожалуйста, отключайте TUN перед обновлением подписки и включайте его снова после завершения обновления.
